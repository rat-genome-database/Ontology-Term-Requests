# Ontology Term Requests (CMO, MMO, XCO)

![Ontology Requests](https://img.shields.io/badge/ontology-requests-blue)
![Issues Welcome](https://img.shields.io/badge/issues-welcome-brightgreen)
![Status](https://img.shields.io/badge/status-active-success)

This repository is used to request **new terms, updates, or improvements** for the following ontologies:

| Ontology | Name |
|--------|--------|
| **CMO** | Clinical Measurement Ontology |
| **MMO** | Measurement Method Ontology |
| **XCO** | Experimental Condition Ontology |

Researchers, curators, and ontology users are encouraged to submit requests to improve ontology coverage and quality.

---

# Submitting a Term Request

To request a new term or modification:

1. Go to the **Issues** tab
2. Click **New Issue**
3. Select **Ontology Term Request**
4. Fill out the template

Providing detailed information helps ontology curators process requests more efficiently.

---

# Request Template (Preview)

When submitting a request, you will be asked to provide information like the following:

| Field | Description |
|-----|-----|
| Ontology | CMO, MMO, or XCO |
| Term Name | Proposed term label |
| Definition | Clear definition of the term |
| Parent Term | Existing ontology term this should fall under |
| Synonyms | Alternative labels (optional) |
| References | Literature or database references |
| Additional Notes | Any other helpful context |

---

# Types of Requests Accepted

This repository accepts requests for:

- New ontology terms
- Updates to existing terms
- Definition improvements
- Additional synonyms
- Hierarchy adjustments
- Clarifications to ontology structure

---

# Review Process

1. A curator reviews the issue.
2. Additional clarification may be requested.
3. Approved terms are added to the ontology.
4. The issue is closed once the request has been processed.

Accepted changes will appear in a future ontology release.

---

# About the Ontologies

### Clinical Measurement Ontology (CMO)

Represents **clinical and physiological measurements** used in biomedical and phenotyping studies.

### Measurement Method Ontology (MMO)

Describes **methods and techniques** used to obtain measurements.

### Experimental Condition Ontology (XCO)

Captures **experimental treatments and environmental conditions** applied in studies.
